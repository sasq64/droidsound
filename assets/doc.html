Droidsound Help

v1.2

The User Interface

Droidsound lets you switch between 3 main screens using the button in the top-right corner;

The Browser

This lets you browse through your music collection.

Tapping a directory, archive or play list will enter it.
Tapping a song will play it.
Press BACK to go back up one level.
Long press BACK to go up to the top level.
Long press a song for more options.
Tap the titlebar to change sort order

The Info Screen

This screen shows information about the current song, and lets you control playback.

For songs that support it, use the progressbar to seek to a desired position in the song
Tapping the Forward / Back buttons will switch sub tunes. If there are no subtunes, it will switch song.
Long pressing Forward / Back will always go to the next / previous song.
Pause and stop should be self-explanatory
Tap the PLUS-sign to add the current song to the active playlist.
Tap SEQ / RND to switch between sequential and random order playback.
Tap CONT to turn on/off continuous playback, meaning the next song will play automatically when the current is done. Note that changing sub tunes will turn CONT off, to avoid switching song when the user may not want to.
Tap W to turn on/off wakelocking, which mean Droidsound forces the screen not to blank.

The Search Screen

This screen shows the latest search results, and also provides a button for searching (besides the one in the main menu). The search screen works similar to the Browser. You can currently only search for song titles when searching the song database.

Handling Playlists

You can add songs to playlists. There is always an active playlist, starting with the Favorites playlist which is created automatically. Long press on another playlist and choose "Set as current" to change the active playlist.

You can add songs, directories and even other playlists to the active playlist.

To add specific (or even all) sub tunes of a song, you need to start playing that song, and then tap the Plus sign on the Info screen.

Music Playback

Droidsound uses a set of plugins to support various music formats - they are listed below. Music is played by a service, and keeps playing in the background while you do other things.

Droidsound is also associated with its supported music files, so you should be able to click links to music from the Web brower, and they should start playing in the background.
All Android browsers are not working correctly in this regard (for instance Galaxy S always downloads all files) and for that reason you can also select to Share a link with Droidsound to get the same result.

Streaming

Droidsound can stream MP3s over the network, including from Shoutcast & Icecast servers. It can also browse remote websites if they are presented in standard Apache dirindex mode.
Here are some ways you can try this out;

  • Click on one of the links in the included Network folder
  • Long press an M3U or link to an MP3 in the webbrowser, and Share it to Droidsound
  • From the Droidsound menu, create a New HTTP-link to a website with music

Ringtones

To create a ringtone from a song, long press the song and select Create Ringtone and choose the desired options.
Default format is WAV 22KHz mono, select HQ for Stereo 44KHz.

Keyboard shortcuts

If you have a hardware keyboard, you can use these shortcuts:
  • [1-9] Subtunes
  • [Z] Previous Song
  • [X] Play/Pause
  • [V] Stop
  • [B] Next Song
  • [N] Previous Subtune
  • [M] Next Subtune
  • [S] Shuffle On/Off
  • [R] Repeat mode
  • [U] Favorite menu
  • [I] Context menu
  • [O] Option menu
  • [P] Settings
  • [F] Search
  • [TAB] Switch view
  • [,] Volume down
  • [.] Volume up
  • [DEL] Goto parent
  • [Q] Quit

Databases

Droidsound provides access for "external" databases. Currently only CSDb is supported. When you enter a database, performing a search will search that database instead of your normal music collection.

Headset & Media Buttons

You can control Droidsound with the Media button on your headset;

Single tap to play/pause.
Squeeze ( > 0.3s) to change song.
Hold for more than 2s to turn speech on/off.

Note that the button will not be activated unless the player is playing or paused.

The Song Database

Droidsound uses a database both for browsing and searching files, so it needs to keep the files on storage in sync with it's database. It does this by scanning for changes each time you start Droidsound (you can also do an explicit scan from the Settings menu).

When changes are detected (an file that has been added, removed or changed) Droidsound locks the database and opens a dialog showing progress. This should only happen when there are actual changes, if you see the dialog each time it probably means you have files that are not detected as songs somewhere in your MODS-directory.

The database (as well as some internal data and config files) can be found in the droidsound/ folder on your SD card.
